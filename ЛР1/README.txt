ЛР1 по курсу "Программная инженерия"
Реализация LRU-кэша должна уметь делать следующие вещи:
1. Поддерживать как ключ (строку), так и приоритет (время добавление);
2. Находить элемент по ключу;
3. Находить и удалять элемент с минимальным приоритетом;
4. Обновлять приоритет узла в сторону увеличения;
5. Вставлять элемент с максимальным приоритетом;
6. Только требование ЛР - удалять элемент по ключу.
Реализация LRU-кэша в ЛР поддерживает эти операции со следующими асимптотиками:
1. _;
2. O(1);
3. O(N);
3. O(1);
5. O(1);
6. O(1).
Можно реализовать всё за log(n), если использовать декартово дерево (treap).
